## Strong's Hebrew & Greek Dictionaries

| Field | Details |
| --- | --- |
| **Source name** | Open Scriptures — Strong's Dictionaries of Hebrew and Greek |
| **Source URL** | https://github.com/openscriptures/strongs |
| **Direct data files** | https://github.com/openscriptures/strongs/tree/master/greek/strongs-greek-dictionary.js · https://github.com/openscriptures/strongs/tree/master/hebrew/strongs-hebrew-dictionary.js |
| **Underlying work** | *Strong's Exhaustive Concordance* by James Strong, S.T.D., LL.D. (1890) |
| **License** | **CC BY-SA 3.0** on the Open Scriptures JSON derivative (see file headers in source repository). The underlying 1890 Strong's concordance text is in the **public domain** in the United States. |
| **Redistribution status** | **Permitted** with attribution and ShareAlike compliance for the Open Scriptures derivative layer. Nomomartyria redistributes normalized JSON derived from the Open Scriptures files. |
| **Attribution requirements** | Credit **Open Scriptures** and **James Strong (1890)**. Retain notice that data is derived from Open Scriptures Strong's dictionaries (CC BY-SA 3.0). |
| **Date checked** | 2026-07-06 |

### Sources explicitly **not** used

- Logos, Bible Hub, Blue Letter Bible, or other copyrighted online Strong's pages (scraping prohibited).
- Any source without a clear public-domain or permissive license statement.
